Conceptual UI

Room 404 is a surreal interface study—constructed like a half-remembered dream, rendered like a controlled glitch. Drawing from abandoned login portals, orbital hotel lobbies, and the quiet tension of retro-futurism, this project merges 3D environments with interface elements that feel strangely familiar.
